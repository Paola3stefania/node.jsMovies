# CLI with Node.js

Software developed in plain node.js with no libraries (js vanilla).

## Installation

To run the custom command you must execute:

```bash
npm install
```

```bash
npm link
```

## Usage

```javascript
[*: Required, **: optional]

command-movie
  get-person
    --id <int> *

  get-persons
    --page <int> *
    --popular *

  get-movie
    --id <int> *
    --reviews

  get-movies
    --page <int> *
    --popular **
    --now-playing **
```
